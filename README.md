# SnapStory
> A public SnapChat story downloader

## Installation

```bash
# clone the repo
$ git clone https://github.com/sdushantha/SnapStory.git

# install the requirements
$ pip3 install -r requirements.txt
```

## Usage
```
usage: snapstory.py [-h] username
```
